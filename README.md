# Projeto EMA - Backend

Este repositório contém o backend do sistema de monitoramento de Estações Meteorológicas Automáticas (EMA).

## Funcionalidades

- Recebimento de dados das estações meteorológicas
- Processamento e armazenamento dos dados
- Disponibilização de API para consumo externo

## Tecnologias

- Node.js
- API REST
